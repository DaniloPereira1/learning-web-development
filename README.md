# 🌐 Web Development Learning Repository | ウェブ開発学習リポジトリ

このリポジトリは、FreeCodeCampの**Responsive Web Design**カリキュラムを通じた学習の進捗を記録するものです。モダンなウェブ開発の強固な基礎を築くことに焦点を当てています。

セマンティックな構造、スケーラビリティ、そして国際化に対応したウェブプロジェクトの作成を通じて、HTMLとCSSの実践的なスキルを習得することを目的としています。

---

## 🎯 目標 (Objectives)

* **技術的基礎:** HTML5とCSS3の強固な基礎の構築。
* **国際化 (i18n):** 多言語（日本語、英語、ポルトガル語）に対応したインターフェースの開発。
* **コードの品質:** セマンティックな構造の適用と、整理された読みやすいコードの維持。
* **ポートフォリオの開発:** 実践的なスキルと開発者としての継続的な成長を示すプロジェクトの作成。

---

## 🛠️ 技術とコンセプト (Technologies & Concepts)

* **HTML5:** セマンティック構造、アクセシビリティ（`alt`、`lang`）、コンテンツの整理。
* **CSS3 (進行中):** スタイリング、レイアウト、レスポンシブデザイン。
* **ローカリゼーション (L10n):** 個別のページとナビゲーションフローによる多言語対応。
* **SVG:** アイコンや視覚要素のためのスケーラブルなグラフィックス。
* **Git & GitHub:** バージョン管理とプロジェクトのドキュメント化。

---

## 🌍 国際化への取り組み (Internationalization Focus)

一部のプロジェクトでは、以下の理由から多言語サポート（日本語、英語、ポルトガル語）を取り入れています。

* 言語ごとに異なるインターフェースへの適応を練習するため。
* さまざまな文脈における可読性と構造を向上させるため。
* グローバルなユーザーに向けたアプリケーションの準備をするため。

---
